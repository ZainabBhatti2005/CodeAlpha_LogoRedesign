# CodeAlpha Logo Redesign 🎨✨

A graphic design project completed as part of the CodeAlpha Graphic Design Internship, focused on a modern logo redesign for a fictional wellness brand named **Nimbus**.

---

## 🚀 Project Overview

This repository contains the visual assets and layout configurations for the Nimbus logo redesign. The project showcases three distinct lockup variations engineered to fit different brand touchpoints, alongside a deliberate design rationale prioritizing modern, clean, and balanced aesthetics suitable for a contemporary wellness brand.

---

## 📂 Included Deliverables

*   **`nimbus_logo_redesign.svg`**: The primary master file containing the vector artwork, featuring:
    *   3 distinct logo lockup variations (Vertical, Horizontal, and Icon-only).
    *   Embedded design rationale detailing typography choices, grid alignment, and the color palette selection.

---

## 🛠️ Tech Stack & Tools

*   **SVG (Scalable Vector Graphics)** - Resolution-independent, clean XML-based graphics optimized for both web and print applications.
*   **Vector Design Software** - (e.g., Illustrator / Figma / Inkscape) utilized to construct precise geometries and balanced padding.
